# アナログ部

## サイト概要

### サイトテーマ

カードゲーム、ボードゲームなどアナログで遊ぶゲームのレビューサイト

### テーマを選んだ理由

デジタル化が進んでいる中、直接会って遊べるアナログゲームを世に紹介し、広めていきたいことから、テーマに選定しました。

### ターゲットユーザ

ゲームが好きなおともだち

### 主な利用シーン

おすすめしたゲームの紹介、遊んでみたいゲームの検索など
